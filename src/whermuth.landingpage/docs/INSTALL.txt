whermuth.landingpage Installation
---------------------------------

To install whermuth.landingpage using zc.buildout and the plone.recipe.zope2instance
recipe to manage your project, you can do this:

* Add ``whermuth.landingpage`` to the list of eggs to install, e.g.:

    [buildout]
    ...
    eggs =
        ...
        whermuth.landingpage
       
* Re-run buildout, e.g. with:

    $ ./bin/buildout
